# Customer Segmentation & Business Impact Dashboard
## Overview

This project presents an interactive Power BI dashboard built on an Online Retail dataset to analyze:

Customer behavior using **RFM segmentation**

Revenue contribution across segments

Customer churn risk

Business impact through revenue concentration

The highlight of this dashboard is a **dynamic dual-view system** using bookmarks, allowing users to switch between:

**Customer Intelligence View**

**Business Impact View**

—all within a single page

## Problem Statement

Businesses often struggle to:

Identify high-value customers

Detect churn risk early

Understand revenue dependency on specific segments

This dashboard solves these problems by combining customer analytics + business insights in one unified interface

Key Features
**Customer View**
RFM-based segmentation (Recency, Frequency, Monetary)

Customer value distribution

Churn risk analysis

Repeat customer behavior

**Business View**
Revenue contribution by segment

Revenue concentration (Pareto analysis)

Revenue at risk from inactive customers

High-value customer identification

**Advanced UX (Highlight Feature)**
Bookmark-based navigation

Dynamic titles and KPI switching

Context-aware insight cards

## Tech Stack
Power BI – Data modeling & dashboarding

DAX – Measures, KPIs, dynamic insights

Power Query – Data cleaning & transformation

Excel – Initial preprocessing

## Data Processing
Handled missing values and outliers

Created RFM metrics

Built segmentation logic

Optimized dataset for performance

## Dashboard Highlights
Single-page design with dual analytical perspective

Interactive filters and slicers

Clean and structured layout

Insight-driven storytelling
